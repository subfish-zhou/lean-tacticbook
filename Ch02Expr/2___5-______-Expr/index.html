3.5. 2.5 构造 Expr🔗

读取 Expr 用模式匹配，构造 Expr 用 mkApp 系列函数：

3.5.1. 手动构造🔗

-- 构造 @Eq Nat a b
let eqExpr := mkApp3 (mkConst ``Eq) (mkConst ``Nat) a b

-- 构造 @Eq.refl Nat a  (即 rfl 的证明项)
let rflExpr := mkApp2 (mkConst ``Eq.refl) (mkConst ``Nat) a

常用构造函数：

| 函数 | 说明 | |------|------| | mkConst n | 常量引用 | | mkApp f a | 单参数应用 | | mkApp2/3/4/5/6 | 多参数应用 | | mkAppN f #[a,b,c] | 数组参数应用 | | mkLambda n bi t body | 构造 λ 表达式 | | mkForall n bi t body | 构造 ∀ 表达式 |

3.5.2. 用引号构造（quotation）🔗

在 TermElabM / TacticM 中，可以用 quotation + elabTerm 从语法直接构造：

let e ← elabTerm (← `(Nat.add 2 3)) none
-- e 是精化后的 Expr

这更方便，但有额外开销（要走一遍 elaboration）。在性能敏感的 tactic 中，手动 mkApp 更好。
